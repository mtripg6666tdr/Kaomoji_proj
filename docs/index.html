みょんかおもじ - Microsoft IME向け顔文字辞書プロジェクト

みょんかおもじとは

「みょんかおもじ」はMicrosoft社の入力ソフト、Microsoft IME向けの顔文字辞書を制作するプロジェクトです。

Microsoft IMEにははじめから顔文字が多数収録されていますが、それでは物足りない人のためのカスタムユーザー辞書を提供します。 収録されている顔文字はmtripg6666tdrがネット上で見つけたものやそれらを組み合わせたものです。
